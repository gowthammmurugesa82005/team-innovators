Project Details

Project Overview

This project focuses on the development of an automated system for detecting cardiac arrhythmias using FPGA technology. By implementing detection algorithms on a Field Programmable Gate Array, we achieve high-speed, real-time processing of ECG signals. The system is designed to identify irregularities in heart rhythm by analyzing the QRS complex, providing a portable and efficient solution for continuous cardiac monitoring without the latency issues of software-based systems.

Problem Statement & Objectives

The primary challenge addressed is the need for immediate, reliable detection of life-threatening heart conditions in a wearable format. Traditional methods often rely on PC-based analysis which isn't always portable, or microcontrollers that may struggle with the computational load of complex filtering.
1. To design a hardware-efficient QRS detection algorithm.
2. To implement the Pan-Tompkins or similar threshold-based logic in Verilog HDL.
3. To ensure the system can distinguish between Normal Sinus Rhythm and various Arrhythmias (like Tachycardia or Bradycardia) in real-time.

Tech Stack Used
Verilog HDL: Used for coding the digital logic, including the FIR filters and the integration-based detection logic.
Xilinx ISE/Vivado: The primary environment for synthesis, implementation, and generating the bitstream for the FPGA.
Spartan-3E FPGA: The target hardware used for physical verification and real-time signal input testing.
MATLAB/Python: Employed for initial algorithm modeling and generating test vectors from the MIT-BIH database to verify the Verilog output.
Features & Results

The project resulted in a functional hardware prototype with the following outcomes:
Digital Filtering: Successfully implemented Low-Pass and High-Pass filters in hardware to remove baseline wander and powerline interference from the raw ECG.
Real-Time Detection: The system identifies R-peaks with high precision, calculating the heart rate (BPM) instantaneously.
Resource Efficiency: The design utilizes minimal FPGA slices, leaving room for additional features like data encryption or multi-lead analysis.
Visual Feedback: Integrated with an LCD or 7-segment display to show the detected heart rate and a status LED to indicate the detection of an arrhythmic event.
